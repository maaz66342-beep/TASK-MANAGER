# 📋 Task Manager API

A production-ready REST API with JWT Authentication, Role-Based Access Control, and React.js frontend.

## 🚀 Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** SQLite (Sequelize ORM)
- **Auth:** JWT + bcryptjs
- **Frontend:** React.js
- **Docs:** Swagger UI

## ✅ Features
- User Registration & Login with JWT
- Role-Based Access (User / Admin)
- Task CRUD with filters & pagination
- Swagger API Docs at `/api/v1/docs`
- Input validation & error handling

## ⚙️ Setup
```bash
cd project/backend
npm install
npm run dev

cd project/frontend
npm install
npm start
```

## 📖 API Docs
`http://localhost:5000/api/v1/docs`
